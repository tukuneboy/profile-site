# GWAT - ITエンジニア ポートフォリオサイト

福岡を拠点としたITエンジニア「GWAT」（代表: 大野清司）の自己紹介・ポートフォリオウェブサイトです。

## 🎯 プロジェクト概要

このサイトは、フリーランスITエンジニアとしてのサービス内容、実績、スキル、プロフィールを紹介し、お客様からのお問い合わせを受け付けるためのプロフェッショナルなポートフォリオサイトです。

## ✨ 実装済み機能

### 1. **レスポンシブデザイン**
- PC、タブレット、スマートフォンに完全対応
- モバイルメニュー（ハンバーガーメニュー）実装
- 各デバイスで最適な表示を実現

### 2. **ナビゲーション機能**
- 固定ヘッダーナビゲーション
- スムーススクロール
- アクティブセクションのハイライト表示
- モバイル対応メニュー

### 3. **セクション構成**
- **HERO**: インパクトのあるトップビジュアル
- **SERVICE**: 5つのサービス内容を紹介
- **PROFILE**: 代表者のプロフィールと保有資格
- **SKILLS**: 技術スキルの一覧
- **WORKS**: 実績紹介（WEBアプリ、業務アプリ、インフラ）
- **CONTACT**: お問い合わせフォーム

### 4. **お問い合わせフォーム**
- 名前、メールアドレス、件名、メッセージ本文の入力欄
- クライアントサイドバリデーション
- メール形式チェック
- 送信完了メッセージ表示
- エラーメッセージ表示

### 5. **アニメーション効果**
- スクロールに応じたフェードインアニメーション
- ホバーエフェクト
- スムーズなトランジション

### 6. **デザイン要素**
- モダンなグラデーション背景
- Font Awesome アイコン使用
- Google Fonts（Noto Sans JP、Roboto）使用
- 統一感のあるカラースキーム

## 📁 ファイル構造

```
.
├── index.html          # メインHTMLファイル
├── css/
│   └── style.css       # スタイルシート（レスポンシブ対応）
├── js/
│   └── script.js       # JavaScript（インタラクティブ機能）
└── README.md           # このファイル
```

## 🚀 使用方法

### ローカル環境での起動

1. プロジェクトディレクトリで `index.html` を開く
2. ブラウザで表示される

### 公開する場合

「Publish」タブから簡単にデプロイできます。

## 🛠️ 使用技術

- **HTML5**: セマンティックなマークアップ
- **CSS3**: 
  - Flexbox / Grid レイアウト
  - CSS Variables（カスタムプロパティ）
  - アニメーション・トランジション
  - メディアクエリ（レスポンシブ対応）
- **JavaScript (Vanilla)**:
  - DOM操作
  - イベント処理
  - Intersection Observer API
  - フォームバリデーション
- **外部ライブラリ**:
  - Font Awesome 6.4.0（アイコン）
  - Google Fonts（Noto Sans JP、Roboto）
